Project Title: Clean Energy Awareness Assistant (SDG 7 Mentor)

Problem Statement:
Addressing the global energy awareness gap and combating misinformation around sustainable infrastructure implementation among students and institutions. Many students lack access to reliable, fact-based guidance on renewable energy transitions.

Detailed Solution Description:
I developed an AI-powered 'Clean Energy Awareness Assistant' web application. It acts as a 24/7 mentor, providing fact-based guidance on solar energy, wind power, and energy efficiency aligned with UN SDG 7 (Affordable and Clean Energy). The platform features an intuitive chat interface for instant queries and an integrated 'Green Quiz' to gamify sustainability education. It focuses on breaking down complex technical concepts into accessible, actionable insights for students.

AI Elements and Tools Used:

Core Engine: IBM BOB (No-Code AI Development Platform).

Logic: Advanced Prompt Engineering & RAG (Retrieval-Augmented Generation) logic.

Frontend: HTML5, CSS3, and JavaScript (Generated via IBM BOB).

Design Philosophy: Responsible AI (Transparency-first design with mandatory disclaimers).

Key Features:

24/7 AI Mentor: Instant educational responses on renewable energy.

Green Quiz: Interactive assessment to reinforce sustainability knowledge.

Responsible AI: Built-in safeguards advising users to consult experts for financial decisions.
